# Simulador de Titulação Ácido-Base

Um simulador interativo de titulação ácido-base construído com HTML, CSS e JavaScript. Este projeto permite visualizar a curva de titulação, o pH em tempo real e a mudança de cor do indicador.

## Funcionalidades

- **Visualização da Titulação**: Representação visual de uma bureta e um gobelé (beaker).
- **Gráfico em Tempo Real**: Curva de titulação (pH vs Volume) gerada dinamicamente com Chart.js.
- **Cálculos Químicos Precisos**:
  - Ácido Forte (HCl) + Base Forte (NaOH).
  - Ácido Fraco (CH3COOH) + Base Forte (NaOH).
  - Cálculo de pH em regiões de tampão e excesso.
- **Indicador Visual**: Simulação de mudança de cor da fenolftaleína (incolor a rosa).
- **Controlos Personalizáveis**:
  - Seleção do tipo de ácido.
  - Ajuste de volume e concentração.
  - Adição manual,

## Como Executar

Basta abrir o ficheiro `index.html` em qualquer navegador web moderno.

## Tecnologias Utilizadas

- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- [Chart.js](https://www.chartjs.org/) para a visualização de dados.
